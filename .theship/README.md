# TheShip AI Workspace

This directory contains specifications and tasks synced from your TheShip board.

## Folder Structure
* `llms.txt` - Configuration guide for local AI code assistants (Cursor, Claude Code).
* `features/` - Subfolders containing active features.
  * `{feature-title}/01_PRD.md` - Product requirements and acceptance criteria.
  * `{feature-title}/02_TASKS.json` - JSON data representing active tasks.
  * `{feature-title}/guide.md` - Implementation details and guides.
  * `{feature-title}/tasks.md` - Main engineering task list and checkbox sync.
  * `{feature-title}/tasks/` - Detailed step-by-step documentation for each individual task.

# Implementation Guide: About Page

## Problem Statement
The current website lacks a dedicated About Page, preventing visitors from quickly learning who runs the site and the company’s basic background. This results in a missing touchpoint for credibility and transparency.

## Success Metrics


## User Stories
- As a visitor, I want to see the company name and founding year on an About Page so I can understand who created the site.
- As an admin, I want to update the company name or founding year without editing code, ensuring quick maintenance.
- As a developer, I want a lightweight component that renders this data and passes automated tests.
